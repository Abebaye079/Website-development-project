## Project Overview

This project is a fully responsive, multi-page website for **Lissan Language School**. It highlights the school’s commitment to teaching Ethiopian languages while providing an immersive cultural experience.  

The website includes the following pages:

- **Home** – Hero section, course highlights, benefits, testimonials, and a final CTA.  
- **Courses** – Detailed course listings with descriptions and highlights.  
- **About** – Mission, team members, teaching approach, and languages offered.  
- **Schedule & Pricing** – Weekly schedule table and course package pricing.

The website is fully responsive, adapting smoothly across desktop, tablet, and mobile devices using media queries.

---

## Features

- Sticky navigation bar with hover effects  
- Hero sections with background images and overlay  
- Responsive course cards with hover animation  
- Information sections with benefits and process steps  
- Team and testimonial grids  
- Achievements section with icons  
- Schedule table and pricing package cards  
- Fully mobile-friendly and tablet-ready  
- Promotes Ethiopian languages in a culturally immersive way

## Technologies Used

- **HTML5** – Semantic structure  
- **CSS3** – Styling and layout  
- **CSS Flexbox & Grid** – Responsive layouts  
- **Media Queries** – Tablet and mobile optimization  
- **Google Fonts** – Plus Jakarta Sans for typography

## How to Run

1. Clone or download the repository.  
2. Open any of the HTML files in a web browser.  
3. Make sure the `Style.css` file and `images/` folder are in the correct paths.  
4. Explore the immersive Ethiopian language courses offered by Lissan Language School.
